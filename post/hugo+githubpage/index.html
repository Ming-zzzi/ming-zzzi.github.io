用Hugo和Github Pages搭建个人博客

@ming_zzz · Jul 4, 2019 · 1 min read · category: Blog

记录使用Hugo和Github Pages建站的基本过程，方便以后浏览。

安装Hugo

hugo是一款由Go语言实现的静态网站快速生成框架。简单、易用、高效、易扩展、快速部署

MacOS系统

  • 在终端利用brew命令安装Hugo
brew install hugo

brew的安装方法在这里

Windows系统

  • 根据官网提供的网站下载对应系统的Hugo安装包，将解压出来的文件复制或移动到自己新建的..\Hugo\bin文件夹中
    • 建议安装带有extended的安装包，否则可能会出现文章中Record 4问题。比如我安装的是hugo_extended_0.55.6_Windows-64bit.zip
  • 将之前解压出来的文件中的hugo.exe添加到环境变量的PATH中

验证

  • 安装完之后运行以下命令，如果显示出hugo版本号则说明安装成功
hugo version

新建网站

  • 运行以下命令将网站存放在自定义文件夹中,如果建立成功则会看到相应提示。
hugo new site myblog #放在电脑根目录下的文件夹myblog中

添加主题

  • Hugo官方提供了许多主题可供选择~主题的详细页面上通常都会有主题的添加方法。
    例如我选择的hugo-nuo主题的安装方法如下：
cd themes #移动到myblog文件夹下的themes文件夹里
git clone https://github.com/laozhu/hugo-nuo

每个主题都可以根据自己的喜好来作相应的修改，例如改变图片、字体、颜色，修改内容模块等等。具体如何修改需要阅读主题的说明文件以及模块代码

创建文档

一般性文档

  • cd到myblog目录下后，在post文件夹下生成新文档
hugo new post/helloword.md #生成名为helloword的md文件

使用VS Code等工具打开helloword文件后可以对内容进行编辑。文档的默认格式可以在myblog/archetypes/default.md下进行修改

说明性文档

  • cd到myblog目录下后生成新文档
hugo new about.md #生成名为about的md文件

具体的页面设置方式请参考hugo-nuo的说明文档

本地预览

  • 运行以下命令后，在浏览器里输入http://localhost:1313/即可查看网站内容
hugo server

启动hugo server之后无需退出，保存好文档之后即可同步预览修改内容，非常方便

部署到Github Pages上

  • 在Github上申请一个账号然后登陆
  • 在首页点击New repository，进入页面后在Repository name下输入username.github.io后点击create repository。创建成功后会得到类似这样一个网址：https://github.com/Githun账号username/Githun账号username.github.io.git记一下之后会用到
    注意username一定要写自己Github账号的username，否则你的网站域名会变得不美观
  • 创建完仓库之后，cd到myblog文件夹目录下输入以下命令
hugo

运行此命令后会生成网站的静态页面并全部存放到myblog文件夹的public目录下。

  • 运行以下命令将public目录下的所有文件push到刚刚创建的repository的master分支里
cd public
git init
git remote add origin https://github.com/Github账号username/Github账号username.github.io.git
git add -A
git commit -m "first commit"
git push -u origin master
  • 之后每次修改完网站内容后只需运行以下命令即可同步更新网站
hugo #一定要先生成静态页面
cd public
git add -A
git commit -m "你的评论"
git push -u origin master

※这里提示一下，可以通过以下命令修改每次评论时的昵称和邮箱，昵称会显示在Github里。如若不修改则是默认该电脑的用户名和邮箱地址。在输完git commit -m "你的评论"命令之后，也会看到这样的提示

git config --global user.name "你的username"
git config --global user.email 123456@example.com

后记

使用Hugo和Github Pages创建网站的过程还是十分方便迅速的，非常推荐大家有空可以建个试试~
